## Description
This project was developed as part of the studies on Clean Architecture, a software development approach that seeks to separate the concerns and responsibilities of the different layers of a system, aiming at the maintenance, scalability and testability of the code.

## Installation
```bash
$ npm install
```

## Copying an environment
```bash
$ cp .env.example .env
```

## Running the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## running with docker
```bash
$ docker-compose up or docker compose up
```
